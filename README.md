# GutPattern: Bi-Modal Dietary Trigger Identification

An AI-powered journaling tool designed to uncover hidden dietary sensitivities using mathematical pattern recognition.

### 🧪 The Math
GutPattern uses a **Bi-Modal Time Decay** algorithm to track:
1. **Immediate Reactions:** 0-6 hours (Exponential Decay).
2. **Delayed Reactions:** 30-42 hours (Gaussian/Bell Curve).
